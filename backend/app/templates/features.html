Run local models, own your data, and connect agents across every device you own. Subconscious is an open-source alternative to ChatGPT, Claude, and Grok — and a packaged alternative to Open WebUI.

Core Architecture

Local-First.
Zero Telemetry.

Your data is exclusively yours. All chats, logs, and settings are stored locally on your device. No centralized authority, no telemetry, no data leaving your machine without your explicit consent.

Fully Offline

Run local models via Ollama with zero internet requirement.

Bring Your Own Keys (BYOK)

Connect cloud models using your own API keys. No intermediary, no markup.

Auto-Updates

Subconscious checks for the latest features automatically — one-button seamless update.

Agentic Threads

Create varied threads and agents for distinct purposes. Assign specific tools and fine-tune context effortlessly — each thread can have its own model, persona, and toolset.

Workspaces
